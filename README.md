# changelog

## 0.0.3

- removed hand drawing and made a norma looking canvas
- keyboard shortcuts
  - l -> line
  - r -> rectangle
  - b -> brush
  - c -> circle
  - 1 -> color 1
  - 2 -> color 2
  - 3 -> color 3
  - cmd/ ctrl + z for 'undo'
  - cmd/ ctrl + shift + z  for 'redo'
  - Global shortcut for toggling the canvas
  - multi-screen canvas
